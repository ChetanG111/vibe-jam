# Vibe Jam (Restart)

A Three.js game built from scratch.

## Tech Stack
- **Three.js**: Core 3D engine.
- **TypeScript**: Type-safe development.
- **Vite**: Ultra-fast build tool.
- **GSAP**: For smooth animations.

## Development
```bash
npm install
npm run dev
```

## Legacy Version
The original React Three Fiber version is archived on the `legacy-r3f` branch.

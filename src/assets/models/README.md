# Shipped glTF/GLB Models

Put small, optimized `.glb` or `.gltf` files here if you want to ship Blender-made assets.

Build-time budgets (see `scripts/check-model-budgets.mjs`):

- Max per file: 250 KB
- Max total across all models: 2 MB

Notes:

- Prefer `.glb` with no textures (or tiny textures) for jam constraints.
- Keep meshes low-poly and reuse materials.
